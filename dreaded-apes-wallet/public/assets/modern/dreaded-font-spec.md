# Dreaded Font — Design Spec

Dreaded Font is the custom display direction for Dreaded Apes: a hybrid of graffiti energy and typewriter structure.

## Style DNA
- Monospaced core for wallet/UI clarity and a mechanical crypto-terminal feel.
- Stencil cuts and worn ink traps for a rugged patched-uniform look.
- Graffiti edges, drips, and exaggerated strokes for headers, stickers, and social assets.
- Best used for wordmarks, headings, badges, banners, and promo copy.
- Avoid long body copy; pair with a clean monospace or geometric sans for UI text.

## Suggested Pairings
- Display: Dreaded Font direction/specimen.
- UI monospace: Space Mono, IBM Plex Mono, JetBrains Mono, or similar.
- Body/UI sans: Inter, Sora, Manrope, or similar.

## CSS Font Stack
```css
--font-display: "Dreaded Font", "Space Mono", "Courier New", monospace;
--font-ui: Inter, Sora, Manrope, system-ui, sans-serif;
--font-mono: "JetBrains Mono", "IBM Plex Mono", "Space Mono", monospace;
```

Note: This package includes the visual font specimen and usage direction, not an installable font file.
